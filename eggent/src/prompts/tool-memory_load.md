# Memory Load Tool

Search persistent memory for relevant information using semantic similarity search.

## Usage

- Provide a descriptive search query
- Results are ranked by semantic similarity
- Higher scores indicate more relevant matches

## When to Use

- Before asking the user for information they may have provided before
- When you need to recall previously saved facts or solutions
- When starting a conversation to check for relevant context
- When working on a task that may have related previous solutions

## Tips

- Use natural language queries for best results
- Be specific in your query to get relevant results
- If no results are found, the information may not have been saved yet
